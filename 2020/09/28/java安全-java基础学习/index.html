1.前言

2.知识笔记

1.数据类型小计

数据类型四种:

1.整型

2.字符串型->是个String类,应该不算基础类型

3.布尔型

4.浮点型

如果超出数值范围的话，会报错的。同时如果是float类型的数值没有带着f，是会报错的。但是改成double类型的话就不会报错了。

2.java基础类型赋值大范围不能赋给小范围

3.this 表示当前类对象

4.关系操作符:

长路与和短路与:(&和&&)

长路与即使第一个表达式是false，也会执行第二个表达式，长路与两边都会进行运算

短路与，如果第一个表达式是false，不会执行第二个表达式。

长路或和短路或:(|和||)

长路或，不管是false还是true，两边都会执行运算

短路或，如果第一个表达式true，就不会执行后面的运算，反之其他情况下全都会进行表达式运算。

异或:(^)

不同返回真，相同返回假:

创建和赋值

6.类和对象

1.重载就是相同的方法，但是有不同的方法的方法参数

但是如果参数很多的话，每次都重载的话，会非常麻烦。因此可以使用可变数量的参数，即可以代表所有方法了

1.this使用来代表当前对象

在构造方法中可以使用构造方法:this(),this(参数)

2.引用和=

赋值：

float f = 1.25f 中的=表示赋值，其中f就是变量。

引用：

hacker t=new hacker();

t是引用,其中=不再是赋值的意思,=表示指向的意思

hacker t=new hacker();这个语句的含义是:

引用t指向hacker对象

每次指向对象,即使指向同一个对象,引用地址都不一样。

3.修饰词

\1. 属性通常使用private封装起来
\2. 方法一般使用public用于被调用
\3. 会被子类继承的方法，通常使用protected
\4. package用的不多，一般新手会用package,因为还不知道有修饰符这个东西

当一个属性被static修饰 的时候,就叫做类属性，又叫到静态属性。

当一个属性被声明成类属性的时候,所有对象都共享一个值。

7.方法

对象方法：又称为实例方法，非静态方法。

必须要有对象，才能调用

类方法：静态方法

可以直接类调用，无需实例化对象，来调用

8.初始化代码块的执行顺序

属性初始化>静态初始化>初始化代码块>构造方法

9.接口与继承

PS:如果想要实现接口的话，必须实现接口上的方法。一个类是可以实现多个接口的，但是继承是不能的。

1.向上转型：子类转父类

只要子类继承了父类的话，子类引用赋值给父类是不需要强制类型转换的。

2.向下转型：父类转子类

子类继承了父类的话，，父类引用赋值给子类需要强制类型转换

3.向上转型：类转换成接口

4.向下转型：接口转换成类

5.instanceof

判断一个引用所指向的对象。

10.重写

子类可以继承父类的对象方法（静态方法不能被继承的），在继承后，我们可以在子类中对该方法重复提供（并修改）。

11.隐藏

子类覆盖父类类方法

13.实例化的小tips

实例化子类，会先调用父类的无参构造方法，然后调用子类的构造方法：

14.super

super调用父类带参的构造方法：super除了可以调用父类的构造方法外，还可以调用父类的属性和方法。

15.抽象类和接口的区别

1.子类只能继承一个抽象类，不能继承多个

2.子类可以实现多个接口

3.抽象类可以定义public、protected、package、private，静态和非静态，final和非final属性，但是接口中声明的属性，只能是public、静态、final。
