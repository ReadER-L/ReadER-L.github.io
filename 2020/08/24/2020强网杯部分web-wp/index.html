2.解题

引入了 ob_start 和 ob_end_clean 函数， ob_start 函数作用是开启缓冲区，将所有输出，例如题目中的 print $result)存入缓冲区，待 ob_end_clean 函数清空缓冲区。我们的目的是要输出变量 $flag ，也就是 ssrf.php 文件的内容。 要得到 $flag ,就需要执行 $result=$GLOBALS[‘flag’] ，然后通过 read 方法输出 $result ，而如果仅一次反序列化，是没办法输出的。所以需要两次反序列化，传入一个数组。
接着就是处理 ob_end_clean ，因为再输出语句是存入缓冲区的，如果 ob_end_clean 执行了，就无法输出了。
所以得想办法让程序在 ob_end_clean 之前报错，而 global $$a 可以进行一次变量覆盖，如果得到 global $this
