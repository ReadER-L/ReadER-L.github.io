Linux基础二

本文最后更新于：2020年12月18日 晚上

​ Linux复习笔记第二章

1.Linux的文件类型

文件名：是文件的唯一标识符。
Linux中文件名遵循以下原则：
➢除/以外都可使用，尽量不使用特殊字符。
➢严格区分大小写字母。
➢文件名尽量代表文件内容和类型。
Linux不强调文件扩展名的作用，但是数据文件通常还是使用文件主名.扩展名格式

1、普通文件（-）
➢文本文件：以ASCII编码形式存储
➢数据文件
➢可执行的二进制程序：以二进制形式存储

2、目录文件

➢简称目录，存储一组相关文件的位置、大小等信息。
➢目录文件由目录项构成，它包括两个部分：文件名和文件号，文件号被称作i节点号 i_number。

3、链接文件
➢硬链接文件：保留所链接文件的索引节点(磁盘的具体物理位置)信息。
➢符号链接文件：只记录所链接文件的路径。

4、设备文件
➢是存放I/O设备信息的文件。Linux中的每个设备都用一个设备文件来表示。所有的设备文件都放在/dev目录下。
➢主要类型是块设备文件(b)、字符设备文件(c)

5、管道文件
➢ 用于不同进程间的信息传递。一个进程将数据写入管道，另外一个进程读取管道的数据。

2.目录文件结构

根目录：\

用户主目：~

工作目录：.
父目录：..

/boot：Linux系统开机信息，其中有系统启动时要安装的内核和模块信息。
/bin：在单用户及多用户下都要用到的一般用户命令。
/dev：存放设备文件，包括块设备和字符设备。
/etc ： 存放系统启动和运行所需的配置和脚本文件，各种应用程序的配置文件和脚本文件。
/home ：一般用户的主目录。
/lib ：系统所需的函数库。
/mnt ：让用户临时挂载其他的文件系统。
/tmp ：存放程序所建立的临时文件。
/usr ：包含大部分的用户命令以及应用程序。
/var ：是系统日志记录分区 。

3.文件操作命令

1.cat命令

格式：cat [选项] 文件列表
功能：显示文本文件的内容。
主要选项说明：
➢-n:在每一行前显示行号

cat三大功能：
1、一次显示整个文件：
cat filename
2、从键盘创建一个文件：
cat > filename
3、将几个文件合并为一个文件：
cat file1 file2 > file

注：输出重定向>配合cat可创建文件，当创建文件时，ctrl+d结束文件输入。

2.more命令

格式：more [选项] 文件
功能：分屏显示文本文件的内容。
注：按Enter键显示下一行内容；
按空格键显示下一屏的内容；
按q键，则退出more命令；
按b键往回翻一页 。

3.less命令

功能：分屏显示文本文件的内容。
less和more命令唯一的不同是less命令可以利用上下键来卷动文件

4.tail/head命令

格式：tail [选项] 文件
功能：
tail显示文本文件的结尾部分，默认显示最后10行。
head命令可以显示文本文件的开头部分
选项说明：
➢-n 数字 指定显示的行数
➢-数字

5.touch命令

格式：touch [选项] 文件
功能：修改文件的时间。更多的时候它会被
用来快速创建一个空文件

6.管道

​ 管道将多个命令连接起来形成一个管道流。
​ 管道流中的每一个命令都作为一个单独的进程运行，前一个命令的输出结果作为后一个命令的输入，从左到右依次执行。
​ 利用“|”符号可实现管道功能。

7.grep命令

格式：grep [选项] 字符串 文件列表
功能：从指定文本文件或标准输出中查找符合条件的字符串
选项：

-v (invert) ：只显示不匹配的行;
-i (ignore) ：忽略大小写;
-n(number)：显示行号;

-c：显示匹配的行数。

字符串正则表达式：
^：匹配行开始，如'^aa'匹配所有以aa开头的行
$：匹配行结束，如’aa$'匹配所有以aa结尾的行

8.find命令

格式：find [路径] [表达式]
功能：从指定路径搜索满足表达式的文件和目录。
主要选项：
-name 文件
-group 组群名
-user 用户名
-type 文件类型(d: 目录; f: 一般文件; l: 符号链接)
-size [+|-]文件大小(c表示字符数;k表示KB)；
+n 是比 n 大；-n 是比 n 小； n 正好是 n
-maxdepth n:最多搜索n层目录

-exec cmd {}:对符合条件的文件执行Linux命令；{}是一个特殊的参数，将被当前文件的完整路径取代；命令必须以\;(与{}之间需要有空格)结束。
-atime n:对文件的最近一次访问是在 n天之前。

9.sort命令

格式：sort [选项] file
说明：对文件中的各行进行排序并显示。
选项：

  • r：按逆序输出排序结果。
  • -k pos1[,pos2]:字段从[posl，pos2]字段从1开始;
  • -t 分隔符：指定字段的分隔符。默认是空格
  • -n：对数字进行排序

10.uniq命令

格式：uniq [选项] file
说明：显示文件内容，显示时删除连续重复的行。
选项：
-d：只显示重复行
-u：只显示不重复行

11.wc命令

格式：wc [选项] 文件
功能：显示文本文件的行数、字节数和单词数。
主要选项说明：
-l（line ）、-w（word）、 -c（character）

12.comm命令

格式：comm [-123] [file1] [file2]
说明：比较两个已排序过的文件，显示其结果。
选项：
-1：不显示只在第1个文件里出现过的列
-2：不显示只在第2个文件里出现过的列
-3：不显示两个文件中相同的行

13.diff命令

格式：diff [选项] 源文件/目录 目标文件/目录
功能：找出两个文件的不同之处，不需要排序

14.cp命令

格式: cp [选项] 源文件/目录 目的文件/目录
功能: 复制文件或目录
选项: -b(backup)、-f(force)、-r或-R(recursive)

15.mv命令

格式：mv [选项] 源文件/目录 目的文件/目录
功能：移动或重命名文件或目录。
选项：-f(force)：强制覆盖
-b：若同名则先备份原文件，再覆盖原文件

16.rm命令

格式：rm [选项] 文件或目录
功能：删除文件或目录。
选项：-f（force）、-r或-R（recursive）
注意：删除目录要用r选项

17.ln命令

格式：ln [选项] 原来文件 链接文件
功能：建立链接文件，默认建立硬链接。
选项：
-b（backup）：若同名备份原文件
-s（symbolic）：建立符号链接

18.mkdir命令

格式：mkdir [选项] 目录
功能：创建目录。
选项: -m 访问权限
-p（parents）:一次性创建多级目录

19.rmdir

格式：rmdir [选项] 目录
功能：删除目录；rmdir只能删除空目录，要删除包含文件或子目录的目录，请用rm –rf进行。

1.文件的访问权限

3.执行权限:对可执行文件而言是允许执行的权限；而对目录来讲是进入目录的权限。

2.与文件权限相关的用户分类

文件所有者（Owner）:建立文件或目录的用户;

同组用户（Group）:文件所属组群中的所有用户;

其他用户（Other）:即不是文件所有者，也不是同组用户的其他所有用户；

[*]超级用户负责整个系统的管理和维护，拥有系统中所有文件的全部访问权限。

2.文件权限相关命令

1.chomd命令

格式1：chmod [选项] 数字模式 文件

格式2：chmod [选项] 功能模式 文件

功能：修改文件的访问权限。
[选项]：
-R：对当前目录下的所有文件与子目录进行相同的
权限改变

2.chgrp命令

格式：chgrp [选项] 组群 文件/目录
功能：改变文件或目录的所属组群。
选项：
-R：对指定目录的所有文件与子目录进行相同更改
-v：显示命令执行过程

3.chown命令

格式：chown 文件拥有者[.组群] 文件
功能：改变文件拥有者，并可修改所属组。
.也可以是：
选项：
-R：对指定目录的所有文件与子目录进行相同更改
-v：显示命令执行过程

4.umask命令

格式：umask [选项] [掩码]
功能：设置限制新文件权限的掩码。umask是从权限中“拿走”相应的位,当文件用命令创建时不能赋予执行权限

5.文件目录的打包和压缩

1.tar命令

格式：tar 选项 归档/压缩文件 [文件或目录列表]
功能：将多个文件或目录归档为tar文件，如果使用相关选项还可压缩归档文件。若是压缩多个文件，文件之间用空格分开。

选项说明：
-c: 创建归档/压缩文件
-C 路径：解压到指定目录下
-r：向归档/压缩文件追加文件和目录
-t：显示归档/压缩文件内容
-u: 更新归档/压缩文件
-x：还原归档/压缩文件
-v：显示执行过程
-z：采用gzip压缩/解压文件
-j：采用bzip2压缩/解压文件
-f：必需选项

2.zip命令

格式：zip [选项] 压缩文件 文件列表
功能：可将多个文件归档后压缩。默认不删除源文件 。
主要选项说明：
-m：压缩后删除源文件
-r：递归压缩目录中的文件

3.unzip命令

格式：unzip [选项] 压缩文件
功能：解压缩扩展名为.zip的压缩文件。
主要选项说明：
-l：查看压缩包里面的文件
-d 目录：指定解压的目录

-o：覆盖同名文件
