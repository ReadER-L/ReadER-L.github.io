保存用户和密码的：

系统日志：PerfLogs是日志目录，系统日志是记录系统中硬件、软件和系统问题的信息，同时还可以监视系统中发生的事件。用户可以通过它来检查错误发生的原因，或者寻找受到攻击时攻击者留下的痕迹。系统日志包括系统日志、应用程序日志和安全日志。

1.c盘根目录常见文件夹

Documents and Settings /用户 通常存储了用户的设置，包括用户文档、上网游览信息、配置文件等数据

包括了Administrators（针对系统管理员的账户）、All Users（针对所有用户）、DefaultUser、用户文件夹等

Application Data:通用应用程序数据文件夹

Local Settings: 保存应用程序数据和历史以及临时文件，可清理

my documents:我的文档

Drivers:部分驱动程序的文件夹

Favorites:收藏夹

Program Files: 应用程序文件夹，一般软件默认都安装在这里，也有一些系统自带的应用程序。在windows 7 系统中64位的用户会多出一个Program Files（x86）的文件夹，这是系统中32位软件的安装目录

Common Files: 共用程序文件夹，用于同系列不同程序软件共同使用或调用数据，比如微软和Adobe的各种软件

ComPlus Applications:微软COM+组件使用的文件夹，删除后可能引起COM+组件不能运行

Program Data: Windows 7 的系统文件夹，放置程序的使用数据、设置等文件，不建议删除

Windows: 存放操作系统主要文件，非常重要

Temp: 存放系统或其他软件临时文件

hiberfil.sys:系统的休眠功能所占用的硬盘空间的文件，不建议删除

pagefile.sys:虚拟内存页面文件，不建议删除

PerfLogs: Windows系统的系统日志和记录

3.window端口（信息收集，目标单侧，服务判断，系统判断，系统角色分析）

常见需要记住的端口：

端口：21服务：FTP说明：FTP服务器所开放的端口，用于上传、下载。

端口：22服务：Ssh说明：PcAnywhere建立的TCP和这一端口的连接可能是为了寻找ssh。

端口：23服务：Telnet说明：Telnet远程登录。

端口：53服务：Domain Name Server（DNS）说明：DNS服务器所开放的端口，入侵者可能是试图进行区域传递（TCP），欺骗DNS（UDP）或隐藏其他的通信。因此防火墙常常过滤或记录此端口。

端口：69/udp：TFTP（trivial file transfer protocol），

端口：80/8080/3128/8081/9080：HTTP说明：用于网页浏览。木马Executor开放此端口。

端口：110：SMTP（simple mail transfer protocol），即E-mail

端口：135服务：Location Service说明：Microsoft在这个端口运行DCE RPC end-point mapper为它的DCOM服务。这与UNIX 111端口的功能很相似。使用DCOM和RPC的服务利用计算机上的end-point mapper注册它们的位置。远端客户连接到计算机时，它们查找end-point mapper找到服务的位置。HACKER扫描计算机的这个端口是为了找到这个计算机上运行Exchange Server吗？什么版本？还有些DOS攻击直接针对这个端口。

端口：443服务：Https说明：网页浏览端口，能提供加密和通过安全端口传输的另一种HTTP。

端口：1524服务：ingress说明：许多攻击脚本将安装一个后门SHELL于这个端口，尤其是针对SUN系统中Sendmail和RPC服务漏洞的脚本。如果刚安装了防火墙就看到在这个端口上的连接企图，很可能是上述原因。可以试试Telnet到用户的计算机上的这个端口，看看它是否会给你一个SHELL。连接到600/pcserver也存在这个问题。

  1. Win 2003远程登录，默认的端口号为3389
  2. Oracle数据库，默认的端口号为1521
  3. MS SQL*SERVER数据库server，默认的端口号为1433/tcp 1433/udp
  4. QQ，默认的端口号为1080/udp

注册表

利用注册表防病毒——

不少计算机系统感染了网络病毒后，可能会在这些注册表中作出修改

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Runonce\

下次开机运行，并且只运行一次的项

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run

开机自动启动的项

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\RunServices

在操作系统启动时就开始运行的程序，优先于run中的程序运行，如果没有也是正常的

我们所使用注册表防病毒的原理在于，就上上面说到的注册表在整个系统中起着核心的作用，任何软件硬件的使用都被注册表调度，所以只要一个病毒可以修改注册表，一定程度上他就控制了这台电脑
