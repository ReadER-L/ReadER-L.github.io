php正则匹配学习

本文最后更新于：2020年6月17日 上午

正则匹配

1.^reader 匹配所有已reader开头的字符串

2.[^a-z] 表示匹配所有以非小写字母开头的字符串

3.reader$ 表示匹配以reader结尾的字符串

4.^reader$ 表示精确匹配reader字符串

5.reader|php|web 匹配reader、php、web中的任意一个字符串

6.php内置通用字符集

7.使用**{}**确定出现重复出现的次数

8.**?**与{0,1}一致

9.*****与{0,1}一致

10.**+**与{1,}一致

正则匹配函数

1.int ereg ( string $pattern , string $string [, array &$regs ] )

ereg(匹配规则，要匹配的字符串，匹配结果存放数组)

该函数会将匹配中的子串存在regs数组中，regs[0]为原字符串，regs[1]为第一个匹配中的字符串，regs[2]为第二个匹配中的字符串。如果省略第三个参数，则匹配中返回true;否则返回false。php5.3以上的版本不支持ereg()函数，而是使用preg_match()函数。

2.explode ( string $delimiter , string $string [, int $limit = PHP_INT_MAX ] ) explode(一个分割用的字符或字符串，原始字符串，最多分割成多少个子串)
